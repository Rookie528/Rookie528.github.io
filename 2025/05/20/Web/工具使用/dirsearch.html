dirsearch

windows下要下载dirsearch包，然后用python解析使用，所以我选择用kali

常用参数说明

参数 说明 示例
​-u​/--url​ 指定目标URL ​-u http://example.com​
​-e​/--extensions​ 指定文件扩展名 ​-e php,html,js​
​-w​/--wordlist​ 使用自定义字典 ​-w /path/to/wordlist.txt​
​-r​/--recursive​ 递归扫描 ​-r​
​--threads​ 设置线程数 ​--threads 50​
​--timeout​ 设置超时时间 ​--timeout 5​
​--random-agent​ 使用随机User-Agent ​--random-agent​
​--proxy​ 使用代理 ​--proxy http://127.0.0.1:8080​

‍

高级用法

1. 使用扩展名过滤

python3 dirsearch.py -u http://example.com -e php,html,js

2. 使用自定义字典

python3 dirsearch.py -u http://example.com -w /path/to/custom_wordlist.txt

3. 递归扫描

python3 dirsearch.py -u http://example.com -r

4. 排除特定状态码

python3 dirsearch.py -u http://example.com --exclude-status 403,404

5. 保存扫描结果

python3 dirsearch.py -u http://example.com -o /path/to/report.txt

‍

实用技巧

  1. 降低扫描速度：添加 --delay=1​ 参数，设置每次请求间隔1秒
  2. 只显示特定状态码：使用 --status-codes 200,301,302​
  3. 强制扫描：使用 --force​ 参数，即使目标返回403也继续扫描
  4. HTTP方法：使用 -m GET,POST​ 指定HTTP方法
